# Ktema: A template manager for lazy people
remember the day when you were working with the terminal and needed to create C, makefile, html and other files. did you wrote basic syntax again and again, instead of focusing on the core thing? didn't you feel irritated as unlike modern code editor, there was no automatic basic syntax adding feature in terminal editors? If yes, ktema is for you which helps you to automatically add basic syntax in your code file easily.
## features
- C17 compatible
- simple design
- simplify boring task
- MIT licensed
- creates a directory to store templates
## commands
### create
- used to create a new template
- used as 'ktema create &lt;new template name&gt; &lt;new template file&gt;'
- &lt;new template name&gt; is the name of the new template
- &lt;new template file&gt; is the file that should be used as template
### delete
- used to delete a template
- used as 'ktema delete &lt;template name&gt;'
- &lt;template name&gt; is name of the template
### update
- used to update a template
- used as 'ktema update &lt;template name&gt; &lt;new template file&gt;' 
- &lt;template name&gt; is the name of the new template
- &lt;new template file&gt; is the file that should replace the previous template file and be used as template
### print
- used to print all templates
- used as 'ktema print'
### write
- used to put a template into a file
- used as 'ktema write &lt;template name&gt; &lt;file&gt;'
- &lt;template name&gt; is the name of the new template
- &lt;file&gt; is the file in which template should be put
